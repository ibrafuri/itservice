Key Components of Windows Security

1.User Account Control (UAC):

Purpose: Prevents unauthorized changes to the OS by requiring admin credentials for tasks needing elevated privileges.

Impact: Reduces the risk of malware making system changes without user consent.

2.Windows Defender Antivirus:

Purpose: Built-in antivirus that offers real-time protection against malware, spyware, and viruses.

Impact: Helps detect and block harmful software, reducing infections.

3.Windows Firewall:

Purpose: Host-based firewall that filters network traffic. Admins can block or allow traffic using rules. Impact: Prevents unauthorized access from external networks. 4.Patch Management (Windows Update):

Purpose: Automates downloading and installing security patches and updates to keep systems secure. Impact: Reduces exposure to known vulnerabilities.

5.BitLocker:

Purpose: Full-disk encryption securing data on hard drives, especially useful for lost/stolen devices. Impact: Protects sensitive data and complies with encryption regulations. 6.Active Directory (AD):

Purpose: Centralized service for managing users, devices, and policies. Integrates with Group Policy for security. Impact: Enhances network security with centralized user management. 7.Windows Event Logs and Auditing:

Purpose: Provides logs for security events like login attempts and file access, helping with monitoring and audits. Impact: Aids in detecting unauthorized access and analyzing security incidents.
